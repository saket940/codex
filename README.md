# InternHub

InternHub is a full-stack internship automation platform built with React + Vite on the frontend and Express on the backend.

## Features

- Front-page internship catalog with admin-managed images.
- Mandatory internship selection during registration.
- Intern dashboard with one unlocked daily task at a time.
- Video-based learning using YouTube iframe links or hosted MP4 files.
- Multiple choice validation before unlocking the next day.
- Automatic certificate issuance once all tasks are completed.
- Admin panel to create and publish new internships with tasks and quiz questions.

## Run locally

```bash
npm install
npm run dev
```

- Frontend: http://localhost:5173
- Backend API: http://localhost:4000
